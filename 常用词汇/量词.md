# 量词

| 量词 | 使用对像                       |
| ---- | ------------------------------ |
| ~人  | 人                             |
| ~台  | 机械、车辆                     |
| ~枚  | 薄平物品：纸、盘子、邮票衬衫等 |
| ~冊  | 书、笔记等                     |
| ~歳  | 年龄                           |
| ~回  | 次数                           |
| ~着  | 衣服等                         |
| ~個  | （类似中文）                   |
| ~本  | 细长物品：瓶、铅笔、伞、香蕉等 |
| ~杯  | 容器内的饮料                   |
| ~匹  | 小动物：猫、狗、鱼、昆虫等     |
| ~頭  | 大动物：牛、马、象等           |
| ~羽  | 鸟、兔                         |
| ~番  | 顺序                           |
| ~足  | 成对的物品：鞋袜等             |
