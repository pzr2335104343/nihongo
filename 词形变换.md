## 一、形容词

### 基本形→连用形（作状语）

- #### 最初に名前を==正しく==書いてください。

- #### 何を聞きたいことが==速く==言って。

  > 形容词作状语时，用形容词的连用形，词尾「い」变成「く」

### 基本形→连接式（连用形+て）

- #### 僕はかっこいく==て==優しいです。

- #### 昨日私は忙しく==て==勉強が忘れります。

  > 形容词　+（形容词/形容动词/动词/句子）时，词尾==い==变为==く== + 接续助词==て==。产生并列或较弱的因果关系。

### 基本形→假定形

> 形容词的假定形是把词尾「い」改为「けれ」：
>
> - ない　→　なけれ
> - 美しい　→　美しけれ
>
> 接续助词「ば」接在动词，形容词，助动词的==假定形==后面，表示以下意义：
>
> 1. 事物的一般性条件，只要A成立，B就成立
> 2. 个人行为或者某个事物的反复出现
> 3. 表示假定。表示假定的句子经常和推测、意志、询问等表达方式一起使用。

## 二、形容动词

### 基本形→で连用形

- #### 人々もとても親切==で==優しかったです。

- #### この店は上海でとても有名==で==、毎日人が大勢来ます。

  > 形容动词　+（形容词/形容动词/动词/句子）时，词尾==だ==变为==で==。产生并列、中顿、因果关系。

### 基本形→に连用形（作状语）

- #### 前の答えを消しゴムで==綺麗に==消してください。

- #### 晩御飯は==静かに==食べてください。

  > 形容动词作状语时，词尾==だ==变成==に==

## 三、动词

> 

### 基本形→ます形连用形

1. #### 五段动词

   > 指う段假名结尾的动词，变换时==词尾假名==变成该行的==い段==。
   >
   > 走る（はしる）　→　走り
   >
   > 洗う　→　洗い

2. #### 一段动词

   > 指以る结尾，并且词尾头一个假名在==い段==（上一段）または==え段==（下一段）的动词，变换时==去掉== ==词尾る==
   >
   > 起きる　→　起き
   >
   > 混ぜる　→　混ぜ

3. #### サ变动词

   > サ变动词为==XXする==，变换时する写成==し==。　例：練習する（れんしゅう==する==）→れんしゅう==し==
   >
   > する　→　し
   >
   > 練習する　→　練習し

4. #### カ变动词

   > カ变动词为==くる==，变换时くる写成==き==。　　例：来る（==くる==）→==き==
   >
   > 来る　→　き

### 基本形→て形连用形

> 动词「て形」本身不具有意义

- #### 五段动词

  > 1. 词尾假名是「つ」、「る」或者「う」的是“	”、词尾变==っ+て==。
  >
  >    - 待（ま）つ　→　==待っ==て           
  >
  >    - 走（はし）る　→　==走っ==て
  > 2. 词尾假名是「ぬ」、「ぶ」或者「む」的是“拨音变”、词尾变==ん+で==。
  >
  >    - 死ぬ　→　==死ん==で
  >    - 読む　→　==読ん==で
  > 3. 词尾假名是「く」或者「ぐ」的是“イ音便”、词尾变==い+て/で==。　
  >
  >    - 書（か）く　→　==書い==て
  >    - 泳（およ）ぐ　→　==泳い==で
  > 4. 词尾为「す」、词尾变成==し+て==。
  >
  >    - 話す　→　==話し==て
  > 5. 特例：行く　→　==行っ==て

- #### 一段动词、カ变动词、サ变动词

  > 和ます形连用形一样  
  >
  > 起きる　→　起き
  >
  > 来る　→　き
  >
  > する　→　し

### 基本形→た形连用形

> 「た形连用形」表示 动词过去式。变形和て形完全相同，只是把て换成た。　死ぬ→死んだ

### 基本形→未然形（意志形、ない形）

> 未然形分为两种、==一种==是词尾变为「ア」段，主要后续否定助动词「ない」；
>
> ==一种==是词尾变为「オ」段、后续助动词「う」、表示意志。

- #### 五段动词

  > 其中「ない」形的「う」段结尾动词，词尾变为「わ」。

  | 简体形 | ない形       | 意志形 |
  | ------ | ------------ | ------ |
  | 話す   | 話さない     | 話そう |
  | 帰る   | 帰らない     | 帰ろう |
  | 洗う   | 洗==わ==ない | 洗おう |

- #### 一段动词

  > 和ます形连用形一样  

  | 简体形 | ない形   | 意志形   |
  | ------ | -------- | -------- |
  | 着る   | 着ない   | 着よう   |
  | 忘れる | 忘れない | 忘れよう |
  | 見る   | 見ない   | 見よう   |

- #### カ变动词

  | 简体形 | ない形 | 意志形 |
  | ------ | ------ | ------ |
  | 来る   | こない | こよう |

- #### サ变动词

  | 简体形 | ない形 | 意志形 |
  | ------ | ------ | ------ |
  | する   | しない | しよう |

### 基本形→假定形

> 动词的假定形是把词尾「う段」改为「え段」
>
> - 五段：行く　→　行け	
> - 一段：起きる　→　起きれ	
> - サ段：する　→　すれ	
> - カ段：来る　→　来れ	
>
> 接续助词「ば」接在动词，形容词，助动词的==假定形==后面，表示以下意义：
>
> 1. 事物的一般性条件，只要A成立，B就成立
> 2. 个人行为或者某个事物的反复出现
> 3. 表示假定。表示假定的句子经常和推测、意志、询问等表达方式一起使用。

### 基本形→命令形

> 表示命令，语气傲慢，男子用的比较多，后边常加上终助词「よ」。
>
> 一段和サ变动词有两种命令形，「よ」用于书面语，「ろ」常用于口语。

- #### 五段动词

  > 词尾的「う段」变为「え段」

  | 简体形 | 命令形 | 例句                           |
  | ------ | ------ | ------------------------------ |
  | 行く   | 行け   | もう時間だから、早く==行けよ== |
  | 書く   | 書け   | ちゃんと==書けよ==             |

- #### 一段动词

  > 词尾る变为ろ或よ

  | 简体形 | 命令形        | 例句               |
  | ------ | ------------- | ------------------ |
  | 見せる | 見せろ/見せよ | 私の力を==見せろ== |
  | 忘れる | 忘れろ/忘れよ | 全部を==忘れろよ== |

- #### サ变动词

  > する→しろ/せよ

  | 简体形 | ない形    | 例句                                  |
  | ------ | --------- | ------------------------------------- |
  | する   | しろ/せよ | 君、さっさと勉強==しろよ==/==せよ==。 |

- #### カ变动词

  > 来る→こい

  | 简体形 | ない形 | 例句         |
  | ------ | ------ | ------------ |
  | 来る   | こい   | 来い、くず。 |

### 基本形→可能形

> 表示具有某种能力或外部条件“有可能/能够”。
>
> ==他动词==构成的可能态，句中的==宾语==常可以构成对象语，此时用「が」表示。
>
> 助动词「られる」「れる」表示==被动、可能、尊敬、自发==
>
> 
>
> ##### 「ら抜（ぬ）き言叶」是一种日本人民新发展出的用于区分可能和被动态的词形。
>
> ##### 表示「一段动词・カ变动词」的可能态时，去掉ら，例如：食べれる　来れる
>
> ##### 这是非正式语言，但不少人用，包括很多年轻人。也是某些地方的日本方言。

- #### 五段动词

  > 未然形+「れる」
  >
  > 表示可能形时，五段动词常发生==约音==：「あ段」假名+「れ」改成当前行「え段假名」

  | 简体形 | 可能形   | 约音   | 例句                           |
  | ------ | -------- | ------ | ------------------------------ |
  | 行く   | 行かれる | 行ける | その時間なら、行けます。       |
  | 読む   | 読まれる | 読める | 漢字が小さく過ぎて読めないだ。 |

- #### 一段动词・カ变动词

  > 未然形+「られる」

  | 简体形 | 可能形/ら抜き言叶 | 例句（ら抜き言叶）   |
  | ------ | ----------------- | -------------------- |
  | 見る   | 見られる/見れる   | 鳥が見れる。         |
  | 来る   | 来られる/来れる   | 仕事なので来れない。 |

- #### サ变动词

  > 未然形+「される」　される是由し和られる约音而来。
  >
  > 表示可能形时，现代日语中，常用「できる」表示而不用「される」

  | 简体形 | 可能形           | 例句                           |
  | ------ | ---------------- | ------------------------------ |
  | する   | できる（される） | お腹が空いた、勉強できないだ。 |

### 基本形→被动形

> 日语中的被动语态大致可以分为直接被动和间接被动两种
>
> 直接被动：是动作直接作用在对象身上、直接受到影响
>
> 间接被动：则是动作没有直接作用在对象身上、间接受到负面影响。
>
> （详：新编第二册第二单元）
>
> 助动词「られる」「れる」表示==被动、可能、尊敬、自发==

- #### 五段动词

  > 未然形+「れる」
  >
  > 表示被动时，五段动词不发生==约音==

  | 简体形 | 可能形   | 例句                 |
  | ------ | -------- | -------------------- |
  | 叱る   | 叱られる | 私は母に叱られました |

- #### 一段动词・カ变动词

  > 未然形+「られる」

  | 简体形 | 可能形   | 例句                               |
  | ------ | -------- | ---------------------------------- |
  | 見る   | 見られる | 彼女は着替えを僕に見られた。       |
  | 来る   | 来られる | 友達に来られて出れなくなりました。 |

- #### サ变动词

  > 未然形+「される」　される是由し和られる约音而来。

  | 简体形 | 可能形 | 例句                                     |
  | ------ | ------ | ---------------------------------------- |
  | する   | される | 遅刻しそうなのに、彼にゲームされている。 |

### 基本形→自发形

> 自发态表示动作不由自主的产生，动词的对象常用「が」表示。变形和==被动态==完全一样。可意为“禁不住、不由得”
>
> 可作自发态的动词大多是和情感有关的动词。
>
> - もう春が近づいたように感じ==られ==ます。
>
> - 雪が降ると、故郷のことが思い出==され==る。

### 基本形→使役形

> 使役句的意义：表示主语让人做某件事。可以表达成”强制、指示、放任、许可“等含义
>
> 使役句的构成：动词未然形 + せる/させる。

- #### 五段动词

  > 未然形+「せる」
  >

  |      | 基本形 | 未然形 | 使役形   |
  | ---- | ------ | ------ | -------- |
  | 五段 | 行く   | 行か   | 行かせる |

- #### 一段动词・カ变动词

  > 未然形+「させる」

  |      | 基本形 | 未然形 | 被役形   |
  | ---- | ------ | ------ | -------- |
  | カ变 | 来る   | こ     | 来させる |
  | 一段 | 見る   | 見     | 見させる |

- #### サ变动词

  > 未然形+「れる」　
  >

  |      | 基本形 | 未然形 | 使役形 |
  | ---- | ------ | ------ | ------ |
  | サ变 | する   | さ     | させる |

> 他动词的使役句，使役者用「は」表示，被使役者用「に」表示。
>
> 自动词的使役句，使役者用「は」表示，被使役者用「を」表示。

### 基本形→被役形

> 动词被役句的意义：表示被使役者被迫或自发做某事。
>
> 动词被役态的构成：动词使役态后续「られる」,五段动词「せら」约音为「さ」。

- #### 五段动词

  > 使役形+「られる」
  >
  > 表示被役形时，五段动词发生==约音==：「せら」约音为「さ」。

  |      | 基本形 | 使役形 | 被役形     |
  | ---- | ------ | ------ | ---------- |
  | 五段 | 行く   | 行かせ | 行かされる |

- #### 一段动词・カ变动词

  > 使役形+「られる」

  |      | 基本形 | 使役形 | 被役形       |
  | ---- | ------ | ------ | ------------ |
  | カ变 | 来る   | 来させ | 来させられる |
  | 一段 | 見る   | 見させ | 見させられる |

- #### サ变动词

  > 使役形+「られる」

  |      | 基本形 | 使役形 | 被役形     |
  | ---- | ------ | ------ | ---------- |
  | サ变 | する   | させ   | させられる |

> 被役句，使役者做补语用「に」表示，被使役者做主语用「は」表示。
